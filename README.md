# FastAPI Product Management System

## Overview

This is a simple Product Management System built using **FastAPI**, **SQLAlchemy**, and **PostgreSQL**. The application provides CRUD (Create, Read, Update, Delete) operations for managing products through REST APIs and a simple HTML/CSS/JavaScript frontend.

## Features

* Add a new product
* View all products
* Get product by ID
* Update product details
* Delete a product
* PostgreSQL database integration
* SQLAlchemy ORM
* FastAPI backend
* Simple frontend using HTML, CSS, and JavaScript

## Tech Stack

### Backend
